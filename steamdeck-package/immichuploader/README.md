<a name="english"></a>
## English
Immich Uploader automatically uploads your Steam Deck screenshots to your Immich instance.

### Configuration
1. **Immich URL:** Your API endpoint (e.g., `http://192.168.1.10:2283/api`).
2. **API Key:** Your personal API key.
